Este projeto é uma lista de exercícios onde praticamos lógica de programação.

## Como Usar

### Requisitos

- .NET SDK (recomendado .NET 8.0 ou superior) para compilação e execução do projeto.

### 1. Clone o repositório.
 
```
git clone https://github.com/AgathaSates/Lista-de-exercicios.git
```
### 2. Abra o terminal ou o prompt de comando e navegue até a pasta de um dos exercícios.
- Exemplo: Exercicio-05
```
cd Lista-de-exercicios/Exercicio-05
```

### 3. Utilize o comando abaixo para restaurar as dependências do projeto.

```
dotnet restore
```

### 4. Execute o projeto

```
dotnet run
```
